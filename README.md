KYRO - Smart Companion and Daily Assistance System

Overview

KYRO is a Smart Companion and Daily Assistance System developed using ESP32 during an Embedded Systems and PCB Design Internship. The system assists users with daily activities through reminders, notifications, voice interaction, and audio-visual feedback.

Features

- Hydration Reminders
- Diet Reminders
- Voice Interaction
- Daily Schedule Assistance
- Health Tips
- Personalized Notifications

Hardware Components

- ESP32 Development Board
- INMP441 Digital Microphone
- OLED Display
- MAX98357A Audio Amplifier
- Speaker

Software Used

- Arduino IDE
- EasyEDA

Applications

- Daily Activity Management
- Personal Reminder System
- Health Awareness Assistance
- Smart Notification System

Internship Project

This project was developed as part of an Embedded Systems and PCB Design Internship conducted at Z00L00P.
